# FINANCIAL-STATE-TRACKER
A Financial State Monitoring Framework for Non-Financial Firms Listed on the Ho Chi Minh City Stock Exchange (HOSE), 2020-2024

**How to Run Locally**

Clone the repository:

git clone https://github.com/duyendm22414/FINANCIAL-STATE-TRACKER.git
cd FINANCIAL-STATE-TRACKER

**Install dependencies:**

pip install -r requirements.txt

**Run the application:**

streamlit run app.py

**Deployment**

This application is designed for deployment on Streamlit Community Cloud.

Deployment requirements:

Public GitHub repository

app.py located at root

requirements.txt included

After deployment, the web application can be accessed via:

https://your-app-name.streamlit.app

(Replace with actual deployment link)

Data Source

Financial statement data were extracted from the FinnPro-X financial database platform.

The dataset includes non-financial firms listed on HOSE from 2020 to 2024.
